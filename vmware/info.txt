Исходник: https://www.if-not-true-then-false.com/2024/debian-vmware-install/

На сайте https://bugmenot.com/ находим расшаренную учётную запись для https://support.broadcom.com/
Авторизуемся на https://support.broadcom.com/ и качаем для нашей операционной системы VMware
Я скачал VMware-Workstation-Full-17.6.4-24832109.x86_64.bundle по ссылке 
https://support.broadcom.com/group/ecx/productdownloads?subfamily=VMware%20Workstation%20Pro&freeDownloads=true
Делаем:
chmod +x VMware-Workstation-Full-17.6.4-24832109.x86_64.bundle
sudo apt update
sudo apt upgrade
sudo apt install linux-headers-$(uname -r) gcc make patch wget pkexec
Запускаем установку
sudo ./VMware-Workstation-Full-17.6.4-24832109.x86_64.bundle
sudo /usr/bin/vmware-modconfig --console --install-all
Это всё, дистрибутив надо качать с сайта.
